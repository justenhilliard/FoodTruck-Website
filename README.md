# Food Truck Website

## Goal
The goal of this project was to build a multi-page HTML website for a food truck. This project helped me practice basic page structure, semantic HTML, navigation, and linking pages together.

## Pages

### Home
The Home page introduces the business and gives visitors a first look at the website.

Food Truck Name: Justen's Yumtastic Chicken  
Tagline: Crispy, Juicy, and Utterly Yumtastic  
Welcome Section: A short welcome message for visitors  
Short Intro to the Business: A brief description of the food truck and what it serves  
Link or Button to the Menu Page: Helps users move easily to the menu

### Menu
The Menu page shows the food items sold by the food truck.

Menu Items with Pictures: Each item can be shown with an image  
Name: The name of each food item  
Short Description: A few details about the item  
Price: The cost of each item

### About
The About page explains more about the food truck.

Story of the Food Truck: How the business started  
Type of Food Served: The style of food offered  
Mission or What Makes It Unique: What makes the truck special compared to others

### Contact
The Contact page gives customers ways to reach the business.

Phone Number: A number customers can call  
Email: A business email address  
Social Media Links: Links to platforms like Instagram or Facebook  
Hours and Location: When the truck is open and where it can be found  

Contact Form:
Name  
Email  
Message  
Submit Button  

## Pages Must Include
Each page in the website should include the following:

Proper HTML Boilerplate: Basic HTML structure with `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`  
Page Title: A title inside the `<title>` tag  
Header: A section at the top of the page  
Navigation Bar: A menu with links to all pages  
Same Navigation on Every Page: Keeps the website consistent  
Links Using `<a>` Tags: Connects each page together  
Working Navigation: Makes sure users can move through every page  
Main Content: The main information for that page  
Footer: A section at the bottom of the page  

## What I Learned
This project helped me get better at creating multiple HTML pages and organizing them clearly. I practiced using semantic tags like `<header>`, `<nav>`, `<main>`, and `<footer>`, and I also learned how important navigation is for making a website easy to use. Overall, this project gave me more confidence with basic HTML.
